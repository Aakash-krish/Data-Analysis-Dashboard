# Insurance Customer & Feedback Analysis

## Project Objective 
This project focuses on analyzing insurance customer data and customer feedback using Power BI to derive actionable business insights.

## Dataset used
- <a href = "https://github.com/Aakash-krish/Data-Analysis-Dashboard/blob/main/InsuranceData.csv">Insurance_Claim_dataset</a>
- <a href = "https://github.com/Aakash-krish/Data-Analysis-Dashboard/blob/main/Insurance%2BCustomer%2BFeedback.xlsx">Customer_feedback_dataset</a>

# Insurance Customer Analysis
## Key Analysis Performed:
1.Total Premium Amount, Claim Amount & Coverage Analysis
2.Claim Status Distribution (Pending, Settled, Rejected)
3.Premium Amount by Policy Type (Auto, Health, Travel, etc.)
4.Claim Amount by Age Group
6.Gender-based customer distribution
7.Policy performance breakdown

## Insights Generated:
1.Identified which policy types generate the highest premium revenue
2.Analyzed age groups contributing to higher claim amounts
3.Compared claim settlement ratios across policy categories
4.Evaluated active vs inactive customer distribution
